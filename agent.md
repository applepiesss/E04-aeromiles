Use the following color palette:

* Primary blue: #4A90E2 (main buttons, key actions)
* Primary hover: #3B7AC4
* Soft blue: #7FB3FF (gradient start / highlights)
* Light blue: #A5C8FF (gradient mid / accents)
* Background tint: #F0F7FF (section background)
* Soft surface: #E0EFFF (secondary buttons / subtle areas)
* Border: #DBEAFE (cards, dividers)
* White: #FFFFFF (main surfaces, cards)
* Text primary: #1E3A5F (headings)
* Text secondary: #3B5F8A (body text)

Use gradients sparingly:

* Hero sections: gradient from #7FB3FF → #A5C8FF → #E0EFFF
* Badges or chips: gradient from #A5C8FF → #DBEAFE
* Avoid using gradients as full page background

Component usage:

* Background: mostly white or very light blue (#F8FAFC or #F0F7FF)
* Cards: white background, subtle blue border (#DBEAFE), soft shadow, rounded corners
* Primary buttons: solid blue (#4A90E2) with hover #3B7AC4
* Secondary buttons: soft blue background (#E0EFFF), darker blue text
* Outline buttons: border #DBEAFE, hover light blue background
* Text: dark blue tones, never pure black
* Shadows: soft and subtle, not heavy

Style direction:

* Clean, minimal, and modern
* Airy and breathable layout with generous spacing
* Rounded components (lg to xl radius)
* Soft shadows and gentle depth
* Slightly premium but still friendly and approachable
* Avoid overly corporate or overly playful style

Consistency keywords:

* soft blue gradient
* clean airline UI
* airy layout
* modern minimal interface
* subtle shadows
* rounded components
* light and breathable
* calm and trustworthy
* not overly saturated

Avoid:

* overly bright or neon blue
* too many gradients everywhere
* dark or heavy UI
* harsh contrast (pure black on white)
* cluttered layout

Goal:
A professional, modern airline interface that feels calm, trustworthy, and slightly premium, with soft blue gradients used as accents.
